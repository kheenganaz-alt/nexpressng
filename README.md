# NexpressNG – Deployment Guide

## 🚀 Deploy to Render (Free Hosting)

### Step 1: Create a GitHub Repository
1. Go to https://github.com and sign in (create account if needed)
2. Click **"New repository"**
3. Name it: `nexpressng`
4. Set to **Public**
5. Click **Create repository**

### Step 2: Upload Your Files
Upload both files to your repository:
- `index.html`
- `render.yaml`

You can drag and drop them directly on GitHub.

### Step 3: Deploy on Render
1. Go to https://render.com and sign up (free)
2. Click **"New +"** → **"Static Site"**
3. Connect your GitHub account
4. Select your `nexpressng` repository
5. Render will auto-detect the `render.yaml` config
6. Click **"Create Static Site"**

### Step 4: Your Site is Live! 🎉
Render gives you a free URL like:
`https://nexpressng.onrender.com`

---

## 🌐 Custom Domain (Optional)
- Buy a domain (e.g. `nexpressng.com.ng`) from any registrar
- In Render → Settings → Custom Domain → Add your domain
- Follow DNS instructions provided by Render
